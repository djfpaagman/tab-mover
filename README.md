# Tab Mover
A really small Chrome extension that allows you to move your tabs with keyboard shortcuts.

## Installation instructions
1. Install the extension directly from the [Chrome Web Store][1]
2. Scroll to the bottom of your 'Extensions' preferences
3. Click 'Keyboard Shortcuts' at the bottom right
4. Set your keyboard shortcuts, I use `Cmd-Shift-O` and `Cmd-Shift-P` because they sit nicely close to tab switching shortcuts `Cmd-Shift-[` and `Cmd-Shift-]`
